# GC Game Dev

Gemini（企画・素材）× Claude Code（実装）で作るゲームプロジェクト

## プロジェクト構成

```
gc-game-dev/
├── index.html
├── css/style.css
├── js/
│   ├── main.js
│   ├── game.js
│   ├── player.js
│   ├── enemy.js
│   └── ui.js
├── assets/
│   ├── images/
│   ├── sounds/
│   └── data/
└── docs/
```

## 開発フロー

1. Geminiが企画・仕様をGoogleドキュメントに記載
2. Claude Codeがドキュメントを読んで実装
3. 動作確認→フィードバック→修正
